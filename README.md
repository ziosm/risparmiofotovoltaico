# Landing Page Fotovoltaico - Deploy Vercel

## 🚀 Deploy Rapido

1. Vai su https://vercel.com
2. New Project → Import
3. Carica questa cartella
4. Configura variabili ambiente:
   - EMAIL_HOST=smtp.gmail.com
   - EMAIL_PORT=587
   - EMAIL_USER=binderecowatt@gmail.com
   - EMAIL_PASSWORD=[password-app-gmail]
   - EMAIL_TO=binderecowatt@gmail.com
5. Deploy!
